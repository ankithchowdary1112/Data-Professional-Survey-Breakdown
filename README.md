# Data-Professional-Survey-Breakdown
🔹 Project Overview

This Power BI project analyzes survey responses collected from data professionals across different departments. The dashboard provides insights into employee ratings, job satisfaction, salary trends, work-life balance, and overall experience within the organization.

🎯 Objective

Analyze employee ratings across departments

Visualize rating distribution using histogram

Compare satisfaction levels between departments

Identify trends in job satisfaction and work environment

Support HR and management with data-driven insights

The dataset contains survey responses from data professionals with the following fields:

Respondent ID

Department

Job Title (Data Analyst, Data Engineer, Data Scientist, etc.)

Experience (Years)

Salary

Work-Life Balance Rating (1–5)

Job Satisfaction Rating (1–5)

Career Growth Rating (1–5)

Overall Performance Rating (1–5)

⚙️ Tools & Technologies Used

Power BI Desktop

Power Query (Data Cleaning & Transformation)

DAX (Data Analysis Expressions)

Data Modeling
